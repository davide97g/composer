# Composer Landing Page

Marketing landing page for Composer - AI-Powered QA Form Agent.

## Repository

GitHub: [https://github.com/yourusername/composer](https://github.com/yourusername/composer)

> **Note**: Update the GitHub URL in `src/components/Footer.tsx` with your actual repository URL.

## Development

```bash
pnpm dev
```

Runs on `http://localhost:5174`

## Build

```bash
pnpm build
```

## Features

- Modern, Mobbin-inspired design
- Responsive layout
- Hero section with CTAs
- Features showcase
- Demo section with placeholders
- How it works guide
- Testimonials
- Stats section
- Call-to-action
- Footer with links

## Assets

### Logo

The logo is located at `public/logo.png` and is used in:

- **Favicon**: Set as the browser tab icon and Apple touch icon
- **Hero Section**: Displayed above the main headline with the "Composer" brand name

To update the logo, replace `public/logo.png` with your new logo file.
